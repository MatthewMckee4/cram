# Cram

A pure-Rust flashcard application with Typst-powered card rendering and SM-2 spaced repetition.

## Features

- Write flashcard content in [Typst](https://typst.app) markup — full math, formatting, diagrams
- SM-2 spaced repetition scheduling
- Live Typst preview in the card editor
- Decks stored as plain TOML files

## Quick Start

```bash
cargo install cram
cram
```
